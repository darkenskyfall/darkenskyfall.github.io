Kesibukan

Saat ini saya bekerja di xenopati.id, salah perusahaan rintisan di kota pahlawan dengan posisi sebagai Pengembang Perangkat Lunak meliputi Pengembangan Aplikasi Seluler berbasis iOS dengan Swift sebagai bahasa pemrograman, lalu pengembangan situs web menggunakan teknologi Laravel dan MySql dan juga kustomisasi situs dengan Wordpress .

Namun beberapa bulan terakhir saya mempelajari bahasa pemrograman Python dan membuat beberapa peralatan untuk menunjang pekerjaan saya dalam memasukkan data secara otomatis atau mengunduh banyak gambar dalam satu kali klik .

Dan juga membuat mini project berbasis situs web menggunakan kerangka kerja Django sebagai media pembelajaran saya dan mungkin akan rilis secepatnya .
