Tentang Saya

Salam Kenal! Saya Dimas Pramudya Sumarsis dari Surabaya, Mulai menjadi pengembang perangkat lunak 4 tahun terakhir sejak duduk di bangku sekolah menengah kejuruan hingga saat ini bekerja di salah satu startup di kota pahlawan .

Pengembangan perangkat lunak meliputi membangun website dengan framework Laravel, membangun aplikasi berbasis iOS dengan Swift hingga uji coba . Untuk saat ini saya tertarik untuk mendalami bahasa pemrograman Python khususnya framework Django untuk membangun website .
