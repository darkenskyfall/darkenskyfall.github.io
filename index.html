Tentang Saya

Salam Kenal! Saya Dimas Pramudya Sumarsis dari Surabaya, Mulai menjadi pengembang perangkat lunak 4 tahun terakhir sejak duduk di bangku sekolah menengah kejuruan hingga saat ini bekerja di salah satu perusahaan rintisan di kota pahlawan .

Di waktu senggang saya biasa untuk menulis dan mengutarakan sesuatu yang berhubungan dengan hidup, teknologi dan pengalaman saya
semoga kamu terinspirasi ;)

Kesibukan

Saat ini saya bekerja di perusahaan rintisan di kota pahlawan dengan posisi sebagai Pengembang Software meliputi Pengembangan Aplikasi Seluler berbasis iOS dengan Swift sebagai bahasa pemrograman, lalu pengembangan situs web menggunakan teknologi Laravel dan MySql dan juga mengerjakan kustomisasi website menggunakan Wordpress .

Namun beberapa bulan terakhir saya mempelajari bahasa pemrograman Python dan membuat beberapa peralatan untuk menunjang pekerjaan saya dalam memasukkan data secara otomatis atau mengunduh banyak gambar dalam satu kali klik .

Dan juga membuat mini project berbasis situs web menggunakan kerangka kerja Django sebagai media pembelajaran saya dan mungkin akan rilis secepatnya .
